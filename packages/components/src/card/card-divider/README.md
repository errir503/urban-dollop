# CardDivider

CardDivider renders an optional divider within a [`<Card />`](../card/README.md).

## Usage

```jsx
import { Card, CardBody, CardDivider } from '@wordpress/components';

const Example = () => (
	<Card>
		<CardBody>...</CardBody>
		<CardDivider />
		<CardBody>...</CardBody>
	</Card>
);
```
